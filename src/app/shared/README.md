# SHARED – Documentación

La carpeta `shared` contiene componentes UI reutilizables.

## Contenido permitido
- Custom components  
- Pipes  
- Directivas  
- Controles de formulario  
- Componentes UI que se usen en 3+ features

## Contenido prohibido
- Servicios  
- Lógica de dominio  
- Modelos de features  

## Custom Components obligatorios
- `custom-table/`  
- `custom-form/`  
- `custom-modal/`  
- `filter-panel/`  
- `page-header/`  

## Estructura esperada
```
shared/
 ├── components/
 │     ├── custom-table/
 │     ├── custom-form/
 │     ├── custom-modal/
 │     ├── filter-panel/
 │     └── page-header/
 ├── directives/
 └── pipes/
```
